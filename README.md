# Google Cloud Dataplex

This is a fork of the official Google Cloud Dataplex repository and contains managed connectivity connectors to import metadata from 3rd party systems to [Google Cloud Dataplex](https://cloud.google.com/dataplex/)

## About this Repository
 * The `managed-connectivity` provides connectors for 3rd party systems

See each directory README for more information.
